# Banpro Javascript mods

Es un pequeño script de demostracion que se creo en vivo por la comunidad de Javascript Nicaragua en el evento de NOV@JSNI que nos permite mejorar la experiencia de usuario de la pagina de Banpro en linea.

### Prerrequisitos

El script corre en el add-on de Google Chrome [Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=es). Por lo tanto, es necesario descargarla e instalarla primero.

### Correr el script

Ejecutar el script es tan facil como visitar el dashboard de la aplicacion, copiar y pegar el codigo del repo y visitar la pagina de Banpro en linea.

Para instrucciones mas detalladas y mas caracteristicas del add-on pueden visitar [Tampermonkey](https://tampermonkey.net/)
